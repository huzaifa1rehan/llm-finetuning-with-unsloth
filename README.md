🚀 LLM Fine-Tuning with Unsloth

This project demonstrates how to fine-tune a Large Language Model (LLM) using Unsloth, enabling faster training, lower VRAM usage, and efficient deployment.

📌 Project Overview

In this repository, we fine-tune an open-source LLM using Unsloth to improve performance on a custom dataset. The workflow includes:

Dataset preparation
Model loading (base LLM)
Parameter-efficient fine-tuning (LoRA/QLoRA)
Training optimization using Unsloth
Saving and testing the fine-tuned model
⚙️ Features
⚡ 2–5x faster training than standard HuggingFace fine-tuning
🧠 Memory-efficient (works on limited GPU VRAM)
🔧 Supports LoRA / QLoRA fine-tuning
📉 Reduced compute cost
📦 Easy model saving and exporting
🧰 Tech Stack
Python 🐍
PyTorch
Hugging Face Transformers 🤗
Unsloth
PEFT (Parameter Efficient Fine Tuning)
TRL (Transformer Reinforcement Learning library)
